Message framing

This module implements framing for MLS messages.

See [MlsPlaintext] and [MlsCiphertext] for details.

Enums

ProcessedMessage

Message that contains messages that are syntactically and semantically correct. [StagedCommit] and QueuedProposal can be inspected for authorization purposes.

UnverifiedContextMessage

Contains an [VerifiableMlsPlaintext] and a Credential if it is a member message. It sets the serialized group context and verifies the membership tag for member messages. It can be converted to a verified message by verifying the signature, either with the credential or an external signature key.
