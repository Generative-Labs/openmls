ProposalQueue

Proposal queue that helps filtering and sorting the Proposals from one epoch. The Proposals are stored in a HashMap which maps Proposal references to Proposals, such that, given a reference, a proposal can be accessed efficiently. To enable iteration over the queue in order, the ProposalQueue also contains a vector of ProposalReferences.

ProposalReference

Reference to a Proposal. This can be used in Commit messages to reference proposals that have already been sent
