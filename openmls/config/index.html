Constants

DEFAULT_KEY_PACKAGE_LIFETIME

This value is used as the default lifetime of KeyPackages if no default lifetime is configured. The value is in seconds and amounts to 3 * 28 Days, i.e. about 3 months.

DEFAULT_KEY_PACKAGE_LIFETIME_MARGIN

This value is used as the default amount of time (in seconds) the lifetime of a KeyPackage is extended into the past to allow for skewed clocks. The value is in seconds and amounts to 1h.
