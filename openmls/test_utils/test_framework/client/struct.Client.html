pub struct Client {
    pub identity: Vec<u8>,
    pub credentials: HashMap<CiphersuiteName, Credential>,
    pub key_store: KeyStore,
    pub groups: RefCell<HashMap<GroupId, ManagedGroup>>,
}
Expand description

The client contains the necessary state for a client in the context of MLS. It contains the group states, as well as a reference to a KeyStore containing its CredentialBundles. The key_package_bundles field contains generated KeyPackageBundles that are waiting to be used for new groups.
