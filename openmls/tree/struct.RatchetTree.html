The ratchet tree.

Fields

ciphersuite: &'static Ciphersuite

The ciphersuite used in this tree.

nodes: Vec<Node>

All nodes in the tree. Note that these only hold public values. Private values are stored in the private_tree.

private_tree: PrivateTree

This holds all private values in the tree. See PrivateTree for details.

7.7. Update Paths

Update the path for incoming commits.

The path contains a public key and encrypted secret value for all intermediate nodes in the path above the leaf. The path is ordered from the closest node to the leaf to the root; each node MUST be the parent of its predecessor.
