Module openmls::group::config[−][src]

Expand description

Configuration module for MlsGroup configurations.

Building an MlsGroupConfig

The MlsGroupConfigBuilder makes it easy to build configurations for the MlsGroup.

use openmls::prelude::*;

let group_config = MlsGroupConfig::builder()
    .use_ratchet_tree_extension(true)
    .build();

See MlsGroupConfigBuilder for all options that can be configured.

Wire format policies

Only some combination of possible wire formats are valid within OpenMLS. The WIRE_FORMAT_POLICIES lists all valid options that can be set.

use openmls::prelude::*;

let group_config = MlsGroupConfig::builder()
    .wire_format_policy(MIXED_CIPHERTEXT_WIRE_FORMAT_POLICY)
    .build();
